# Honda_PowerBi_dashboard
# 🏍️ Honda Motorcycle Sales Dashboard | Power BI

A comprehensive Power BI dashboard for analyzing Honda motorcycle sales performance. This dashboard provides insights into sales, profits, customer ratings, finance, insurance, and regional sales trends through interactive visualizations.

## 📌 Project Overview

This Power BI dashboard helps analyze Honda motorcycle sales data by providing key business insights. It enables users to monitor sales performance, customer satisfaction, financial metrics, and regional trends for better decision-making.

---

## ✨ Features

- 📈 Interactive Power BI Dashboard
- 🏍️ Motorcycle-wise Sales Analysis
- 💰 Gross Profit & Net Sales Analysis
- ⭐ Customer Rating Analysis
- 💳 Payment Mode Distribution
- 🛡️ Insurance Amount Analysis
- 💵 Finance Amount Analysis
- 🌍 State-wise Sales Distribution
- 📅 Monthly Gross Profit Trend
- 🛒 Sales Channel Analysis
- 🔍 Interactive Filters (Bike Selection)
